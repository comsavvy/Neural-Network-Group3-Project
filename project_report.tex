\section{Introduction}
Object detection is a fundamental task in computer vision that involves identifying and localizing objects within an image. In this project, we implemented the YOLOv5 algorithm to perform object detection on the Pascal VOC 2012 dataset, which contains images of 20 different object classes. The goal was to train a model that can accurately detect and classify objects in the images, and to evaluate its performance using standard metrics.
